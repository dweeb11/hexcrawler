# ADD: The Waning Light

## Art Direction
ASCII-glyph terminal aesthetic. Dark background, monospace glyphs, muted palette. The atmosphere comes from text and restraint, not illustration. Every visual element is a Unicode character rendered on Canvas 2D.

## Palette
- Background: `#0a0a0a` (near-black)
- Text/glyphs: `#c0c0c0` (silver)
- Borders: `#333` (dark gray)
- Resource green: `#4a9`
- Warning amber: `#da4`
- Danger red: `#d44`
- Searing gradient: `░▒▓█` with warm tones
- Fog of war: reduced alpha on unrevealed hexes

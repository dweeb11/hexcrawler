# The Waning Light MVP Milestone

## Playtest Checklist

- [ ] Game renders the starting settlement and log panel.
- [ ] `Q/W/E/A/S/D` movement works and reduces Supply on Push.
- [ ] `R` rest and `F` forage enter camp flow and change resources.
- [ ] Encounters can trigger on tagged hexes and resolve choices.
- [ ] The Searing advances over time and consumed hexes become dangerous.
- [ ] Game over triggers on health loss, hope loss, or the Searing catching the player.
- [ ] Encounter data loads from `/api/encounters` or falls back to seed data.
- [ ] Admin panel loads at `/admin.html` and can list/create/update/delete encounters with an API key.
